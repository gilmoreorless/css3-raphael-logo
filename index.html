CSS3 Raphaël Logo

Why? Because I was bored and wanted to do something tricky with CSS3 – and re-rendering the logo of a javascript vector graphics library without using javascript or SVG seemed like a wonderfully pointless way to spend a Sunday evening.

Firefox and Safari are the best browsers to view this in – Chrome used to work fine until one of its many updates changed its border rendering style. Now Chrome seems to stretch and thin out borders with multiple radii defined.

The main things I learned about CSS3 from this:

  1. WebKit doesn't handle border radius on a rotated element properly – there are sometimes distinctive gaps betweeen the curves and straight borders.
  2. -webkit-background-clip should be set to "padding-box" to prevent an element's background colour from bleeding out just past its border.
  3. Firefox's and Chrome's rendering of partial borders with radius measurements tapers the borders to a point while Safari keeps a continuous border width. Chrome used to work in same way as Safari but seems to have switched to Firefox's rendering style (with some extra stretching thrown in).
